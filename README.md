# Windows Terminal Randomizer

Randomize a Windows Terminal profile's font and color scheme, then launch that profile.

## Quick Start

Requirements:
- Windows 10/11 with Windows Terminal
- PowerShell 7+ recommended

Clone and run:
```powershell
git clone https://github.com/kicvincent/terminal-randomizer.git
cd terminal-randomizer/scripts
.\randomize.ps1 -Profile "Alpine"
